# Интересные места

Проект состоит из двух частей:

- `Vite + React` фронтенд
- `Node.js` backend без внешних зависимостей с хранением данных в JSON

## Где хранятся данные

- `data/users.json` — профили и серверные сессии
- `data/places.json` — места, автор места, статусы модерации
- `data/reactions.json` — лайки и дизлайки пользователей
- `data/settings.json` — сезонная тема сайта для будущей админки

## Команды

```bash
npm run dev
npm run build
npm start
```

## Погода OpenWeatherMap

Для погодного блока нужен API-ключ OpenWeatherMap в переменной окружения:

```bash
OPENWEATHER_API_KEY=your_key_here
```

Backend сам запрашивает погоду по координатам места и отдает ее во фронтенд через `/api/weather`.
Во фронтенде показывается:

- текущая температура
- иконка погоды
- скорость ветра
- раскрывающийся прогноз на ближайшие 3 часа

## Порты

- `npm run dev` — фронтенд Vite на `http://127.0.0.1:4322`
- `npm start` — production server на `http://127.0.0.1:4323`
