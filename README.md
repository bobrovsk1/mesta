# Интересные места

Проект состоит из двух частей:

- `Vite + React` фронтенд
- `Node.js` backend без внешних зависимостей с хранением данных в JSON

## Где хранятся данные

- `data/users.json` — профили и серверные сессии
- `data/places.json` — места, автор места, статусы модерации
- `data/reactions.json` — лайки и дизлайки пользователей
- `data/settings.json` — сезонная тема сайта для будущей админки

## Загрузка фото

- сервер сохраняет загруженные фото в `uploads/places`
- папка создается автоматически
- изображения сжимаются через `sharp`
- формат хранения: `webp`
- максимальная ширина: `1600px`
- качество `webp`: `75`
- в `data/places.json` хранится только путь вида `/uploads/places/file.webp`
- `uploads` не попадает в git

## Погода Open-Meteo

Backend сам запрашивает погоду по координатам места через Open-Meteo и отдает ее во фронтенд через `/api/weather`.
API-ключ и переменные окружения для погоды не нужны.
Во фронтенде показывается:

- текущая температура
- описание погоды
